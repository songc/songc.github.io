Hbase使用和配置

配置

  1. hbase-env.sh 设置Java_home

  1. back-masters 设置 备用master

一行对应一个主机名，这个可有可无的。

  1. hbase-site.xml

  1. regionservers

设置运行regionservers服务的节点，一行对应一个主机名，可以有多个（一般和master不在同一个节点）

使用

  1. 启动 start-hbase.sh 启动hbase。
  2. 关闭 stop-hbase.sh 关闭hbase。
  3. 启动和关闭Thrift 服务（这个是用于其他编程语言的接口，不需要的话，没必要用） bin/hbase-daemon.sh stop thrift. To start Thrift on a node, run the command bin/hbase-daemon.sh start thrift.

注意事项

  1. web ui 默认端口。 16010 (1.01版本之前是60010)
  2. RegionServer 管理Region，和对Region进行分区。
  3. Region只有在RegionServers的机器上存在。
