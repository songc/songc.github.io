Windows 软件推荐

发表于 2018-11-14
  1. Typore. 一款markdown编辑器。开源，免费， 颜值高，功能强大。操作简单。我最喜欢的一个markdown编辑器。它同时支持OS X， Windows，和Linux3大操作系统。安装Pandoc后，它还可以支持Word，RTF，LaTeX，PDF，HTML等格式的导出。官网

picture

  1. Virgo, Windows上的一个开源的虚拟桌面的软件，仅仅8KB，详细的介绍可以看这篇文章， 下载地址, 和Win10自带的虚拟桌面类似，但有些不同。

  2. VS Code，微软出品的一个开源跨平台的编辑器，提供了非常的插件。功能强大，可以通过安装不同的插件增强功能。支持JavaScript， TypeScript 以及Node.js。 可以通过安装不同的插件，提供对Java， C++， Python等其他语言的支持，整合了Git操作。界面友好。 ​

  3. everything. Windows上的文件搜索利器，有些文件不记得放到哪里了，只知道名字，使用everything搜索，马上找到，如果你还懂得正则表达式的话，你只记得文件的大概名字，也可以的。搜索速度及其的迅速快捷。

  4. snipaste. 免费Windows截图+贴图软件，自带方便的标注等功能，自动检测界面元素区域,十分方便好用，而且可以用于桌面取色。

  5. Ditto，Windows 上一款剪贴板管理器。保存多个剪贴板历史记录。Ctrl + C 复制，Ctrl + V 粘贴，然后多个剪贴记录直接切换。

  6. SumatraPDF 一个比较纯粹的开源PDF阅读器，支持PDF, ePub, MOBI, CHM, XPS, DjVu, CBZ, CBR等格式，小巧，只有8M大小。而且直接格式也比较多。

SumatraPDF

  1. PotPlayer 视频播放器，启动速度快，免费，无广告，颜值也很高。功能强大，支持各种格式。

potplayer

  1. Gliffy Diagrams Chrome 浏览器的一个APP，用于画图，简单好用，缺点可能就是无法导出透明背景的图片了。使用起来非常的简单方便，也十分的小巧。

Gliffy

  1. 7-Zip 开源压缩软件，免费无广告，不流氓。国内的压缩软件好像都是在这个基础上改的。

  2. JetBrains 全家桶 编程IDE，不多介绍了，界面高大上，功能强大。支持各种编程语言。而且学生可以免费使用。

    Jetbrains
