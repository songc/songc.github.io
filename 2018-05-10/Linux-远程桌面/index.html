  1. Windows 在官网下载exe文件后，按照提示安装就可以了。完成后需要重启电脑。

  2. Linux 系统的安装

TAR.GZ version

  • Click on Download and save the TAR.GZ file in the /usr directory.
  • Extract the TAR.GZ archive by running from command line: $ sudo tar zxvf nomachine_6.0.78_1_x86_64.tar.gz
  • Install NoMachine by executing: $ sudo /usr/NX/nxserver –install
  1. 该软件的功能比较完善，操作也简单。多个平台都支持。支持Windows，Linux，Mac，iOS, Android等。
