Java Web 远程连接Hadoop问题以及一些解决办法

远程连接Hadoop时出现以下错误错误

原因: 本机缺少Hadoop环境,安装hadoop的环境,本地机器上没有安装hadoop环境，也没有设置HADOOP_HOME环境变量。

解决办法：可以去github下载一个hadoop的环境包地址,下载后解压，然后在电脑上添加HADOOP_HOME的环境变量。添加解压的后的bin目录到HADOOP_HOME环境变量中。就可以解决问题了，不用担心版本问题。这个好像是通用的。反正我是2.7.2的版本可以用，网上也说是可以通用。并没有亲自一一测试。
